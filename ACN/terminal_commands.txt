   	                               
   	                               TERMINAL COMMANDS
   	                            ----------------------
   	                            
   	                            
1)pwd (Print Working Directory)

   Description: Print the current working directory path.
   Syntax: pwd
         
2)whoami

   Description: Print the current username.
   Syntax: whoami
 
3)clear

   Description: Clear the terminal screen.
   Syntax: clear            
          
4)ls (List)

   Description: List directory contents.
   Syntax: ls [options] [directory]       
  
   Options
   
    4.1) ls -a : Lists all files including hidden files
    4.2) ls -l : Long listing format, displaying detailed information about files
    4.3) ls -lh: Human-readable sizes (e.g., 1K, 2M) in the output of -l.
    4.4) ls -lt: Sorts files by modification time, newest first.
    4.5) ls -lr: Reverse the order of the sort to get reverse alphabetical results or the              
                 oldest entries first
    4.6) ls -R : List files recursively (including all subdirectories)
    
5)cd (Change Directory)
   
    Description: Change the current working directory.
    Syntax: cd [directory]
    
    cd .. :- move back to the last directory
    cd ~  :- move back directly to the home directory
    cd/dir1/dir2:- move to dir2 in single step
     
6)mkdir (Make Directory)

   Description: Create a new directory.
   Syntax: mkdir [directory_name]
   
7)rmdir (Remove Directory)

   Description: Remove an empty directory.
   Syntax: rmdir [directory_name]
   
8)cp (Copy)

   Description: Copy files or directories.
   Syntax: cp [options] source destination
   
7)mv (Move)

  Description: Move (or rename) files or directories.
  Syntax: mv [options] source destination
  
8)m (Remove)

  Description: Remove files or directories.
  Syntax: rm [options] file1 [file2 ...]
  
9)touch

  Description: Create an empty file or update the timestamp of an existing file.
  Syntax: touch [filename]
  
10)cat

     to create a file and to write to it
     after writing enter ctrl+d to save it
